# Rules Audit Report (Archive)

Статус: архивный аналитический документ, не является нормативным источником правил.
Дата актуализации: 2026-04-24.

## Что использовать как источник истины

1. `.cursorrules` — операционные правила агента и приоритеты исполнения.
2. `.antigravity/rules.md` — короткий архитектурный манифест и навигация.
3. `.antigravity/db_protocol.md` — каноничные правила по БД и транзакциям.
4. `.antigravity/styles_and_ui.md` — каноничные UI-стандарты.
5. `.agent/workflows/*.md` — процедурные сценарии выполнения задач.

## Назначение этого файла

- Хранить исторический контекст аудита и решений по унификации.
- Не дублировать действующие правила из файлов выше.
- Не использовать как самостоятельную инструкцию для выполнения задач.
